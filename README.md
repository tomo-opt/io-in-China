# 中国国际组织索引

“中国国际组织索引”是一个基于静态网页搭建的公开检索站点，用于展示纳入本站口径的国际组织及相关驻华机构信息。本站当前主要提供地图总览、机构列表和使用说明三个页面，便于用户从空间分布、结构化字段和分类检索等角度进行浏览与查询。

---

## 一、页面结构

### 1. 地图总览
地图总览页以中国地图为基础底图，按城市聚合展示已收录机构的分布情况。用户可以：
- 使用关键词检索机构名称、城市、属性、行动领域、官网等字段；
- 按属性、行动领域、成立年份、所在城市进行筛选；
- 点击地图上的城市点位或标签，在右侧展开该城市下的机构信息；
- 在右侧展开区域继续检索和筛选当前城市中的机构。

### 2. 机构列表
机构列表页用于展示本站当前已收录的全部机构，支持：
- 关键词检索；
- 按属性、行动领域、成立年份、所在城市进行多条件筛选；
- 分页浏览；
- 卡片视图与列表视图切换；
- 展开查看机构介绍与参考资料。

### 3. 使用说明
使用说明页用于解释本站的页面功能、字段含义、检索方式、筛选逻辑和机构录入依据。后续如网站功能更新或口径调整，也会优先在该页补充说明。

---

## 二、当前字段说明

本站当前主要展示以下字段：

- 中文名：机构中文名称
- 英文名：机构英文名称
- 属性：机构基本类型或性质
- 行动领域：机构主要工作方向或议题分类
- 成立年份：机构或相关实体的成立时间信息
- 所在地 / 所在城市：机构在中国境内的主要落地城市或办公城市
- 官网：机构官方网站或权威公开页面
- 微信公众号：机构公开微信公众号信息（如有）
- LinkedIn：机构 LinkedIn 页面（如有）
- 机构介绍：对机构背景、职能或在华设置情况的简要说明
- 参考资料：用于支持该机构条目录入与后续核验的基础来源

---

## 三、如何检索与筛选

### 1. 关键词检索
本站支持任意字段的模糊检索。用户可以直接输入机构名称、城市、属性、行动领域或官网信息进行搜索。

### 2. 条件筛选
当前支持以下筛选项：
- 属性
- 行动领域
- 成立年份
- 所在城市

筛选支持单项选择，也支持多项组合筛选。用户可以根据自身需要逐步缩小结果范围。

### 3. 浏览方式
在机构列表页中，用户可以根据阅读习惯选择：
- 卡片视图：适合快速浏览和阅读机构简介
- 列表视图：适合横向比较字段信息

---

## 四、机构录入依据

本站当前机构信息整理遵循“公开、可核验、可追溯”的原则。录入时优先参考：
- 机构官方网站
- 驻华办公室页面
- 正式公开名录
- 年报、白皮书或机构公开材料
- 权威数据库或其他可追溯来源

在机构名称、属性、城市、官网和基础说明等字段上，本站尽量采用能够被公开核验的信息表述。部分字段的细化口径和分类标准后续仍会继续完善。
