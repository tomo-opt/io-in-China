使用说明

本站为“中国国际组织索引”公开检索页面，旨在以地图可视化与结构化列表两种方式， 呈现纳入本站口径的国际组织及相关驻华机构信息，便于浏览、筛选、查询与后续研究使用。

一、本站包含哪些页面

1. 地图总览

以中国地图为底图，按城市聚合展示机构分布。用户可通过关键词检索、条件筛选、 点击城市点位或文字标签，查看该城市已收录机构的集中信息。

2. 机构列表

以全量列表形式展示本站已录入机构。支持关键词检索、多条件筛选、分页浏览， 并支持“卡片视图 / 列表视图”切换，便于不同场景下的信息查找与比对。

3. 使用说明

说明本站功能结构、检索与筛选方式、字段口径，以及机构收录与信息整理的基本依据。 后续如有更新，也会优先在本页补充。

二、如何使用“地图总览”

  1. 在顶部检索框中输入机构名称、城市、属性、行动领域、官网等关键词，可进行模糊检索。
  2. 使用“属性 / 行动领域 / 成立年份 / 所在城市”四类筛选器，可进行单选或多选筛选。
  3. 点击地图中的城市红点，或点击城市名称标签，可展开右侧机构详情区域。
  4. 右侧展开区域中，可继续进行关键词检索与局部筛选，仅查看当前城市下符合条件的机构。
  5. 点击“重置”可恢复当前页面的默认检索状态。

三、如何使用“机构列表”

  1. 左侧可通过关键词检索快速查找目标机构。
  2. 可按“属性 / 行动领域 / 成立年份 / 所在城市”进行多条件组合筛选。
  3. 支持分页浏览，并可切换每页显示数量。
  4. 支持“卡片视图”与“列表视图”切换，适用于概览浏览或横向比对。
  5. 在卡片或列表中，可展开查看机构介绍与参考资料。

四、当前筛选项分别代表什么

字段 含义说明
属性 用于区分机构的基本性质，例如政府间组织、非政府间组织及其他相关类型。
行动领域 用于概括机构的主要工作方向或议题范围，是本站进行主题分类展示的核心字段之一。
成立年份 用于展示机构或相关驻华实体的成立时间信息；具体口径后续可根据数据源继续细化说明。
所在城市 用于展示该机构在中国境内的主要落地城市或办公城市，并在地图页中用于城市聚合展示。
官网 用于展示机构官方网站或权威公开页面链接。
机构介绍 用于简要说明机构性质、职能、背景或在华设置情况。
参考资料 用于记录机构条目的基础来源或辅助核验材料，便于后续复核与更新。

五、机构收录与录入依据

本站当前收录工作以公开、可核验的信息源为基础，优先参考机构官网、驻华办公室页面、正式公开名录、 年报、权威数据库及其他可追溯资料，对机构名称、类别、所在城市、官网与基础说明进行整理。

机构是否纳入本站，以及纳入后的字段如何表述，当前遵循“可公开核验、可明确识别、可相对稳定归类”的原则。 部分口径与分类标准后续还将继续完善，因此若个别字段存在需要补充或修订之处，后续会持续更新。
